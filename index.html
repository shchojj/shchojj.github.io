特征组合 (Feature Crosses)

标签（空格分隔）： 机器学习速成课程

特征组合是指两个或多个特征相乘形成的合成特征。特征的相乘组合可以提供超出这些特征单独能够提供的预测能力。

特征组合：将非线性学习整合到线性学习器中。

线性学习器可以扩展到大规模数据集上。

特征组合 (Feature Crosses)：对非线性规律进行编码

特征组合是指通过将两个或多个输入特征相乘来对特征空间中的非线性规律进行编码的合成特征。“cross”（组合）这一术语来自 cross product（向量积）。

虽然$w_3$表示非线性信息，但您不需要改变线性模型的训练方式来确定$w_3$的值。

  • 将两个特征的值相乘形成的特征组合,[A×B]
  • 将五个特征的值相乘形成的特征组合。[A×B×C×D×E]
  • 对单个特征的值求平方形成的特征组合。[A×A]

特征组合 (Feature Crosses)：组合独热矢量

表示 (Representation)

标签（空格分隔）： 机器学习速成课程

特征工程：从原始数据中提取特征称之为特征工程。（75%的时间）

  1. 具有非零值，并且在数据集中出现多次。
  2. 具有明确的意义。
  3. 不随时间发生变化。（由上游模型而来，而该上游模型可能发生变化，希望能够从模型中获取具有定义规范的恒定语义的值。）
  4. 不采用不理性的离群值。
    [a]为特征设置上限或者转换特征。
    [b]分箱技术（独热编码）。

了解数据

  • 可视化：绘制直方图，从普遍到不普遍排列。
  • 调试：重复样本？缺失值？离群值？数据与信心中心一致？训练数据与验证数据相似？
  • 监控：特征分位数、样本数量随着时间推移有无变化？

表示 (Representation)：特征工程

将原始数据映射到特征。

特征矢量：组成数据集中样本的浮点值集。

特征工程指的是将原始数据转换为特征矢量。

机器学习模型期望样本表示为实数矢量。

矢量构建方法：为每个字段衍生特征，然后将它们全部连接到一起。

整数和浮点原始数据不需要特殊编码。

映射字符串值

特征工程将字符串值转换为数字形式：

  1. 为要表示的所有特征的字符串值定义一个词汇表。
  2. 使用该词汇表创建一个独热编码，用于将指定字符串值表示为二元矢量。在该矢量（与指定的字符串值对应）中：
  • 只有一个元素设为 1。
  • 其他所有元素均设为 0。
    该矢量的长度等于词汇表中的元素数。

映射分类（枚举）值

分类特征具有一组离散的可能值。
将分类特征编码为枚举类型或表示不同值的整数离散集。

机器学习模型通常将每个分类特征表示为单独的布尔值。
简化某个值可能属于多个分类这种情况。

表示 (Representation)：良好特征的特点

  1. 避免很少使用的离散特征值。
    良好的特征值应该在数据集中出现大约 5 次以上。这样一来，模型就可以学习该特征值与标签是如何关联的。也就是说，大量离散值相同的样本可让模型有机会了解不同设置中的特征，从而判断何时可以对标签很好地做出预测。
    如果某个特征的值仅出现一次或者很少出现，则模型就无法根据该特征进行预测。因为每个值只使用一次，模型无法从中学习任何规律。
  2. 最好具有清晰明确的含义。
    混乱的数据（而不是糟糕的工程选择）会导致含义不清晰的值。
  3. 不要将“神奇”的值与实际数据混为一谈。
    良好的浮点特征不包含超出范围的异常断点或“神奇”的值（如“不存在”用-1表示）。
  • 一个特征只存储质量评分，不含神奇值。
  • 一个特征存储布尔值，表示是否提供。
  1. 考虑上游不稳定性

表示 (Representation)：清理数据

挑出坏样本并加工可以挽救的样本。
**

缩放特征值

**
缩放是指将浮点特征值从自然范围（例如 100 到 900）转换为标准范围（例如 0 到 1 或 -1 到 +1）。
如果某个特征集只包含一个特征，则缩放可以提供的实际好处微乎其微或根本没有。不过，如果特征集包含多个特征，则缩放特征可以带来以下优势：

  • 帮助梯度下降法更快速地收敛。
  • 帮助避免“NaN 陷阱”。模型中的一个数值变成 NaN，并且模型中的所有其他数值最终也会因数学运算而变成 NaN。
  • 帮助模型为每个特征确定合适的权重。如果没有进行特征缩放，则模型会对范围较大的特征投入过多精力。
  1. 将 [最小值，最大值]以线性方式映射到较小的范围。
  2. 计算每个值的 Z 得分。

处理极端离群值。

离群值这个大尾巴，对数缩放可稍稍缓解这种影响，将特征值限制，存在个小峰值，缩放后的特征集现在依然比原始数据有用。

分箱

类似于积分，小段领域内近似线性。

  1. 以整数作为分箱边界。
  2. 每隔特定值拆分一次分箱边界。
  3. 分位数分箱。

清查

不良样本

  • 遗漏值。
  • 重复样本。
  • 不良标签。
  • 不良特征值。

从数据集中移除问题数据，从而“修正”不良样本。

不良数据直方图是一种用于可视化集合中数据的很好机制。还有其他统计信息：

  • 最大值和最小值。
  • 均值和中间值。
  • 标准偏差。

了解数据

  • 记住预期的数据状态。
  • 确认数据是否满足（不满足）这些预期。
  • 仔细检查训练数据是否与其他来源的数据一致。

良好的机器学习依赖于良好的数据。

验证 (Validation)

标签（空格分隔）： 未分类

每次迭代时，我们都会对训练数据进行训练并评估测试数据，并以基于测试数据的评估结果为指导来选择和更改各种模型超参数

基于给定测试集执行评估的次数越多，不知不觉地过拟合该测试集的风险就越高。

“调整模型”指的是调整您可以想到的关于模型的任何方面，从更改学习速率、添加或移除特征，到从头开始设计全新模型。该工作流程结束时，在测试集上获得最佳效果的模型。

将数据集划分为三个子集（如下图所示），可以大幅降低过拟合的发生几率：

验证集评估训练集的效果，然后，在模型“通过”验证集之后，使用测试集再次检查评估结果。

  1. 在验证集上获得更好的效果。
  2. 使用测试集再次检查该模型。

训练集和测试集 (Training and Test Sets)

发表于 2018-04-01

训练集和测试集 (Training and Test Sets)

标签（空格分隔）： 机器学习速成课程

测试集是用于评估根据训练集开发的模型的数据集。

测试集满足两个条件：

  1. 规模足够大，可产生具有统计意义的结果。
  2. 能代表整个数据集。换言之，挑选的测试集的特征应该与训练集的特征相同。
  • 训练集越大，模型学习效果越好。
  • 测试集越大，评估指标越准确，置信区间越窄。

绝对不要对测试数据进行训练。

泛化 (Generalization)

发表于 2018-04-01

泛化 (Generalization)

标签（空格分隔）： 机器学习速成课程

泛化是指模型很好地拟合以前未见过的新数据（从用于创建该模型的同一分布中抽取）的能力。

奥卡姆剃刀定律的形式化：
模型越简单，良好的实证结果就越有可能不仅仅基于样本的特性。

  • 模型的复杂程度。
  • 模型在处理训练数据方面的表现。

三个基本假设：

  1. 独立同分布（i.i.d.）。
  2. 分布平稳（不随时间变化）
  3. 从同一分布中抽样：训练集、验证集和测试集。

知晓假设前提，密切关注各项指标，违法假设时能够立即知道。

过拟合模型在训练过程中产生的损失很低，但在预测新数据方面的表现却非常糟糕。

过拟合是由于模型的复杂程度超出所需程度而造成的。

机器学习的基本冲突是适当拟合我们的数据，但也要尽可能简单地拟合数据。

机器学习的目标是对从真实概率分布（已隐藏）中抽取的新数据做出良好预测。

训练集、测试集。

测试集上表现是否良好是衡量能否在新数据上表现良好的有用指标，前提是：

  • 测试集足够大。
  • 不会反复使用相同的测试集来作假。

降低损失 (Reducing Loss)

发表于 2018-03-31

如何降低损失

迭代方法

不断迭代，直到总体损失不再变化或至少变化极其缓慢为止。这时候，我们可以说该模型已收敛。

梯度下降

凸形问题只有一个最低点；即只存在一个斜率正好为 0的位置。这个最小值就是损失函数收敛之处。

  • $(y-y’)^2$相对权重$w$和偏差$b$的导数表明样本的损失变化
  • 降低损失的方向上反复取小步（负梯度步长）
  • 梯度是偏导数的矢量；它可以让您了解哪个方向距离目标“更近”或“更远”。

学习速率

学习速率：步长。
超参数是编程人员在机器学习算法中用于调整的旋钮。

  1. 学习速率过低：费时。
  2. 学习速率过高：永远在U形曲线的底部随意弹跳。
  3. 金发姑娘学习速率：损失函数的梯度较小，则可以放心地试着采用更大的学习速率。

权重初始化

  • 凸形：权重从任意位置开始，只有一个最低点。
  • 非凸形：有多个最低点，很大程度上取决于初始值。

SGD 和小批量梯度下降法

批量指的是用于在单次迭代中计算梯度的样本总数。

  1. 超大批量，则单次迭代就可能要花费很长时间进行计算。
  2. 批量大小越大，出现冗余的可能性就越高。
  3. 冗余可能有助于消除杂乱的梯度，但超大批量所具备的预测价值往往并不比大型批量高。
  • 随机梯度下降法：一次抽取一个样本。
  • 小批量梯度下降法：每批包含 10-1000 个样本。

线性回归

线性回归

标签（空格分隔）： 机器学习速成课程

线性回归

  1. 数据绘制成图表，了解下数据的分布
  2. 绘制一条直线来近似地表示这种关系。
    $y’$ 标签（理想输出），$w_1$特征1权重，$x_1$特征（已知输入），$b$偏差（截距）。
  3. 用新的$x_1$推断（预测）$y’$。
  4. 多个特征来表示更复杂的模型。

训练与损失

训练模型：通过有标签样本来学习（确定）所有权重和偏差的理想值
经验风险最小化：检查多个样本并尝试找出可最大限度地减少损失的模型。
损失：对于单个样本而言模型预测的准确程度的一个数值。
训练模型的目标：从所有样本中找到一组平均损失“较小”的权重和偏差。

平方损失（$L_2$ 损失）：一种常见的损失函数.

$= the square of the difference between the label and the prediction$
$= (observation - prediction(x))^2$
$= (y - y’)^2$

均方误差 (MSE):每个样本的平均平方损失.

  • $(x,y)$：样本.
    $x$：模型预测时的特征集。
    $y$：样本标签。
  • $prediction(x)$: 权重*特征集$x$+偏差的函数
  • $D$：多标签样本集$(x,y)$
  • $N$: $D$中样本个数。

机器学习主要术语

发表于 2018-03-31

机器学习主要术语

(监督式）机器学习:通过学习如何组合输入信息来对从未见过的数据做出有用的预测.

标签:要预测的事物，即简单线性回归中的 y 变量。

特征:输入变量，即简单线性回归中的 x 变量。

样本：数据的特定实例：x。

  1. 有标签样本：同时包含特征和标签。用来训练模型。
    labeled examples: {features, label}: (x, y)

  2. 无标签样本：包含特征，但不包含标签。即：

    unlabeled examples: {features, ?}: (x, ?)

模型:定义特征与标签之间的关系。
生命周期两个阶段：

  • 训练：创建或学习模型。向模型展示有标签样本，让模型逐渐学习特征与标签之间的关系。
  • 推断：将训练后的模型应用于无标签样本。用训练后的模型来做出有用的预测 (y’)

回归与分类

  • 回归模型：预测连续值。
  • 分类模型：预测离散值。

标签：量化的指标。
特征：可量化的。

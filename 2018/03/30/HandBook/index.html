hexo n == hexo new # 写文章

更换主题
1、安装：在博客根目录下执行git clone https://github.com/iissnan/hexo-theme-next.git themes/next。
2、启用：修改博客根目录下的_config.yml配置文件中的theme属性，将其设置为next。
3、更新：在themes/next目录下执行git pull。（暂时不需要）
4、\themes\next_config.yml修改主题配置。
